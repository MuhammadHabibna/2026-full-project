# Maze Generator & Solver

Visualisasi algoritma pembuatan dan penyelesaian labirin menggunakan Pygame.

## Fitur
1.  **Generation**: Menggunakan algoritma *Recursive Backtracker* untuk membuat labirin yang sempurna.
2.  **Solver**: Menggunakan *DFS (Depth First Search)* untuk mencari jalan dari pojok kiri atas ke kanan bawah.
3.  **Visual**: Animasi real-time proses pembuatan dan pencarian jalan.

## Cara Menjalankan

1.  Pastikan Python terinstall.
2.  Install library:
    ```bash
    pip install pygame
    ```
3.  Jalankan:
    ```bash
    python maze_solver.py
    ```

Nikmati animasinya!
